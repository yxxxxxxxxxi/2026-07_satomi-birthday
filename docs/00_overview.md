# 概要

- 旅行名: 2026年7月_理実誕生日旅行
- 日程: 2026-07-18 から 2026-07-21 の 3泊4日
- 行き先: 広島、宮島、尾道、出雲、松江、玉造温泉
- 目的: ふたりで無理なく移動しながら、景色とごはんと温泉をちゃんと楽しめる誕生日旅にする

## 旅の空気感

- あわただしく詰め込まず、移動のしんどさを残さないこと
- 7月の暑さを避けつつ、景色のいい時間帯を上手く使うこと
- 食事と宿で「来てよかった」と思える余白をつくること
- 誕生日らしい小さな特別感を、自然に一日の中へ入れること

## いま考えていること

- 絶景と温泉の比重を少し高めにして、夏でも気持ちよく過ごせる流れにする
- 屋外が続きすぎないよう、暑い時間帯は屋内やカフェ休憩を差し込みやすくしておく
- 山陰側は移動時間が長くなりやすいので、見る場所を絞ってゆったり動く
